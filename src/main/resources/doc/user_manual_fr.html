Installer et exécuter FullSWOF-transfer_UI

Vous devez disposer de Java version 1.8 ou supérieure pour pouvoir exécuter FullSWOF-transfer_UI.

L'équipe de développement de FullSWOF utilise la version 11 de Java pour vérifier le bon fonctionnement de FullSWOF-transfer_UI. Ainsi, nous garantissons son utilisation avec la version 11 de Java.
Pour vérifier la version de Java qui est installée sur votre machine, tapez la commande suivante dans un terminal :

java -version

Il n'est pas nécessaire de disposer de FullSWOF_1D pour lancer FullSWOF-transfer_UI.

Pour lancer FullSWOF-transfer_UI en ligne de commande, placez-vous dans le dossier contenant FullSWOF-transfer_UI.jar (sans décompresser le fichier) et lancez la commande :

java -jar FullSWOF-transfer_UI.jar

Sur certains systèmes, il est possible de lancer l'application par un clic droit sur l'archive jar, en l'ouvrant avec Java.


Préférences et aide

Lorsque vous avez lancé l'interface à l'aide de la ligne de commande ci-dessus, l'application s'ouvre. Par défaut, la langue de l'application correspond à la langue de votre ordinateur. Elle contient une barre de menu horizontale. On retrouve le menu Préférences où vous pouvez choisir votre langue. Choisissez entre français et anglais, puis veillez à relancer l'application, sans quoi le changement de langue ne sera pas fait.

Toujours dans la barre de menu horizontale, vous retrouverez un onglet Aide. Vous avez alors à votre disposition ce manuel d'utilisation de l'interface, ainsi que des informations concernant FullSWOF-transfer_UI.


Les fonctionnalités

Importer un fichier de paramètres

Pour réutiliser des données à partir d'un fichier, vous devez utiliser la commande 'Ouvrir un fichier de paramètres'. Cette commande tentera de lire les données issues d'un fichier formaté comme le fichier généré transfer.txt.

Renseigner les paramètres

Vous pouvez aussi partir de rien et créer vous-même le fichier transfer.txt. FullSWOF-transfer_UI vous permet d'indiquer aisément les paramètres de configuration nécessaires pour lancer FullSWOF. Il suffit de renseigner les différents champs. Certains champs utilisent des valeurs entières, et d'autres des décimaux.

Pour certains champs, on vous demande un nom de fichier. Sélectionner le bouton parcourir '...', puis choisissez le fichier adapté à ce champ.

De plus, lorsque vous changez d'onglet puis revenez dans l'onglet précédent, les paramètres ne sont pas perdus. Vous êtes libre de remplir les champs dans l'ordre que vous souhaitez.

Gestion des erreurs

Lorsqu'un champ n'est pas rempli, il apparaît en rouge. Vous pouvez alors le remplir, la couleur rouge disparaîtra.

Si un champ est mal rempli (par exemple, si un champ attend un entier et que vous remplissez ce champ par une chaîne de caractères) alors ce champ indiquera une erreur avec la couleur rouge.

Si vous essayez de générer le fichier transfer.txt alors qu'il y a des erreurs ou des champs non remplis, l'application vous indiquera une erreur et ne générera pas le fichier.

Génération du fichier

Dans le cas où tous les champs sont remplis correctement, vous trouverez dans la barre de menu un onglet Fichier, dans lequel se trouve un bouton Générer. Le fichier est généré ; attention, si le fichier existe déjà, il sera écrasé (un message vous préviendra).
